## Getting Started

1. Clone the repository:

```sh
   $ git clone https://github.com/unsignedint64/uwu-lang.git
```

2. Navigate to the src/ directory:

```sh
  $ cd uwu-lang/src/
```

3. Compile and run the code:

```sh
  $ g++ main.cpp -o uwu
  $ ./uwu
```
